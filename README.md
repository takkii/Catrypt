### Cryptman ( 暗号化・複合化ツール ) → OSS project Catrypt (暗号化・複合化ツール)

[apache codec](https://commons.apache.org/proper/commons-codec/download_codec.cgi)

```markdown
上記リンクからダウンロード(commons-codec-1.14-bin.tar.gzなど環境にあわせる)

使っているIDEでライブラリ登録する。

IntelliJの場合

プロジェクト構造 → プロジェクト設定 → ライブラリー → + を押す→apache codecを登録
```

### 実行

```markdown
git clone git@github.com:takkii/Cryptman.git

IntelliJの場合

プロジェクトをインポート

Cryptmanを選択、OKを継続、完了ボタンを押す
```
